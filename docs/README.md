# AUTOBAGS

> **Your bags, on autopilot.**

AUTOBAGS is an AI-powered Solana memecoin trading agent built on [Bags.fm](https://bags.fm). Sign up, deposit SOL, and let the bot do the work — scouting trending tokens, scoring them for safety and momentum, and executing swaps automatically, 24/7.

***

## What makes AUTOBAGS different?

| Feature | Description |
|---|---|
| 🤖 **AI-driven** | Scores every token across 6 signals before buying |
| 🔐 **Self-custodial option** | Export your private key at any time |
| ⚙️ **Configurable** | Basic risk presets or full manual control |
| 💰 **Fee sharing** | Built on Bags.fm — transparent on-chain fees |
| 🛡️ **Secure** | AES-256-GCM encrypted keys, 2FA on all sensitive actions |
